## [ts.yayujs.com](http://ts.yayujs.com/)

可能是中国最好的 TypeScript 入门到进阶系统教程 

## 介绍

TypeScript 系列文章由官方文档翻译、重难点解析、实战技巧三个部分组成，涵盖入门、进阶、实战，旨在为你提供一个系统学习 TS 的教程，全系列预计 40 篇左右。目前正在持续更新中！
